# test/

Unit and widget tests. Structure mirrors `lib/` and `bin/`.

- **engine/** — Pure Dart tests for core game logic.
- **providers/** — Provider state tests.
- **services/** — Service layer tests.
- **ui/** — Widget tests.
- **mocks/** — Shared mock/fake implementations.
- **goldens/** — Golden image reference files.
- **failures/** — Golden test failure diffs (gitignored).
